# YouTube Downloader (MP3 / MP4)

A simple GUI application to download YouTube videos as **MP4** or extract audio as **MP3** using Python. Built with **Tkinter** and **pytubefix** for reliable YouTube downloads.  

---

## Features

- Download YouTube videos in MP4 format.  
- Extract audio from videos as MP3 (requires FFmpeg for proper conversion).  
- Choose download folder using a simple browse dialog.  
- GUI context menu for easy copy/paste in the link field.  
- Supports both MP3 and MP4 downloads in one go.  

---

## Notes

- MP3 files are initially downloaded in YouTube's audio format (usually WEBM/AAC).  
- To convert properly to MP3, **FFmpeg** is recommended.  
- The app uses **pytubefix**, a maintained fork of pytube, for reliable downloads.
